# AI Security Distilled

Technical blog on AI security — agent threats, defense patterns, and practical threat models distilled from academic research.

**Live site:** https://copilot-autogent.github.io/ai-security-blog/

## About

This blog synthesizes academic AI security research into actionable analysis for practitioners building and deploying AI agent systems. Topics include prompt injection, tool-use exploitation, MCP security, multi-agent threats, and defense patterns.

## Tech Stack

- [Astro](https://astro.build/) — static site generator
- GitHub Pages — hosting
- GitHub Actions — CI/CD

## Development

```bash
npm install
npm run dev      # local dev server
npm run build    # production build
```
